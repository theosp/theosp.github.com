    Welcome

    Hi, I'm Daniel. Computer programming is my core competence, my work and one of the things I enjoy the most doing. In my work I strive to keep the delicate balance between stability and agility. I'm always eager to expand my horizons with new chalenges and technologies.

    My current project

    I currently work as the head web developer for an ambitious startup named Razoss that aims to develop a new platform for services that runs above any modern web browser.

    The main techonlogies I use in my day to day work these days are: Python, Javascript, NodeJS and Google App Engine.
